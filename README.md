# API REST Mínima de Login con FastAPI

Este proyecto implementa una API REST básica utilizando **FastAPI** y **SQLModel**. Su objetivo principal es demostrar la estructura de una API de autenticación validando credenciales contra una base de datos embebida ("quemada" en el código), sin la complejidad de sistemas como JWT o manejo de sesiones.

## Requisitos

* Python 3.10 o superior.
